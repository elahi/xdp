R and RStudio

Before the first class please read through the computer setup instructions that walk you through how to set up your computer to run R and Rstudio. Prep for Wednesday
Install R and Rstudio on your computer (if you haven’t already!), and read the following website:
https://datacarpentry.org/R-ecology-lesson/00-before-we-start.html

When you get to the part about the working directory - create a new folder on your desktop called biohopk-174. We will work from within this folder on Wednesday. In this folder, create an Rstudio project (biohopk-174.Rproj), as well as the following folders:

  • data
  • data_output
  • docs
  • figs
  • scripts

If you are having trouble installing RStudio please seek help from Robin or Nicole
