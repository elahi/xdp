Robustness to assumptions

  1. Observations are independent ***
  2. Variance is ~equal in all \(k\) groups **
  3. Data are normally distributed in all \(k\) groups *

What if 2 & 3 cannot be met?

Try nonparametric method based on ranks: Kruskal-Wallis test.

Life after ANOVA

Planned comparisons (a priori)

  • You planned a comparison before doing the experiment
  • You can do the specific pairwise comparison, no matter the outcome of the ANOVA
  • Same assumptions as ANOVA, but not as robust

Unplanned comparisons (post hoc)

  • If you reject the \(H_0\), can test for all pairwise differences
  • Tukey-Kramer (i.e., Tukey’s HSD) test uses a modified t-distribution to correct for multiple tests
    • conservative test

Planned comparisons

A priori, we were particularly interested in the knee treatment vs control groups.

\[ \begin{aligned} SE = \sqrt{MS_{error} (1/n_1 + 1/n_2)} \\ \end{aligned} \]

Use t-distribution with \(N - k\) degrees of freedom.

ms <- summary(aov1)[[1]]$"Mean Sq"; ms
## [1] 3.6122459 0.4955445
se_pooled <- sqrt(ms[2] * (1/dat_summary$n[3] + 1/dat_summary$n[1])); se_pooled
## [1] 0.3643283

Planned comparisons

x_diff <- dat_summary$mean[3] - dat_summary$mean[1] # knee - control
N <- dim(dat)[1]; k <- length(unique(dat$treatment))
t_value <- x_diff / se_pooled; t_value
## [1] -0.07401095
pt(q = t_value, df = N - k, lower.tail = FALSE)
## [1] 0.5291123

If you do > 1 planned comparison, consider adjusting \(\alpha\), e.g., using Bonferroni correction. But you can do no more than \(k - 1\) planned comparisons!

Unplanned comparisons

Fixed vs. random effects

Model 1 ANOVA: fixed effects

  • groups are predetermined
  • if you did the study again, you would use the same groups
  • you wish for more replication within groups (larger \(n\)), but not more groups (\(k\))

Model 2 ANOVA: random effects

  • groups are chosen randomly
  • if you did the study again, you would use a different sample of groups
  • you wish for more groups (larger \(k\))

Variance components

Variation within groups

  • \(MS_{error}\) is the best estimate of \(\sigma^2\)
  • variance among repeat measurements of femur length

Variation among groups

  • assumed to be distributed normally with
    • grand mean \(\mu_A\) (mean of group means)
    • variance \(\sigma_A^2\) (variance of group means)

Estimating \(\sigma_A^2\) with \(s_A^2\)

This method assumes equal sample size within groups

\[ \begin{aligned} s_A^2 = \frac{MS_{groups} - MS_{error}}{n \text{ (within groups)}} \\ \end{aligned} \]

ms <- summary(aov1)[[1]]$"Mean Sq"; ms
## [1] 0.002463833 0.000356000
var_A <- (ms[1] - ms[2]) / 2; var_A
## [1] 0.001053917

Repeatability

Repeatability: the fraction of the summed variance that is present among groups

\[ \begin{aligned} \text{Repeatability} = \frac{s_A^2}{s_A^2 + MS_{error}} \\ \end{aligned} \]

var_A / (var_A + ms[2])
## [1] 0.7475028
